# AI Feedback Pipeline

## Project Overview
This project is an automated pipeline designed to process and analyze feedback using Artificial Intelligence.

## Technologies Used
* **Python**
* **FastAPI**
* **OpenAI API**
* **Google Generative AI**
* **Pydantic**

## Setup Instructions
To run this project locally:
1. Clone the repository.
2. Install the required dependencies: `pip install -r requirements.txt`
3. Set up your environment variables (API keys).
4. Run the application using FastAPI.
   
## API Documentation
Once the server is running, you can access the interactive API docs at:
`http://127.0.0.1:8000/docs`

## Configuration
Before running, ensure you have the following environment variables set:
* `OPENAI_API_KEY`: Your key for AI processing.
* `DATABASE_URL`: Your connection string for the database.
